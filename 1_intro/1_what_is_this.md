# What is this?

[Lecture](https://www.youtube.com/watch?v=jlPTGdxhPKE&list=PLADa89BQAJkZe4mtun0H8oO70QrVD-KH0&index=1)

It's a course in computer games programming (duh!), focusing on the core technical concepts that arte common to most games in general. It's about the "engine level" stuff that makes games work under-the-hood, and about learning how to build your own.

# What is this not?

It's not a course about game design, or art, or production.
And, it's not about learning how to use someone else's engine either (i.e. Unity, Unreal, etc...).
